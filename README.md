# Parser

Desktop-приложение для автоматической синхронизации входящих писем и Google-таблиц.

## Быстрый старт

1. Установите Python 3.11+ и создайте виртуальное окружение.
2. Установите зависимости:

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

3. Подготовьте локальные параметры:
   - скопируйте `.env.example` в `.env`;
   - укажите `MAIL_*` и `SPREADSHEET_*`;
   - поместите файл сервисного аккаунта Google (`credentials.json`) в локальную директорию (файл не должен попадать в Git).

4. Запуск:

```bash
python main.py
```

## Тестирование

```bash
python -m unittest tests/test_unit.py -v
```

## Что делает приложение

- Подключается к почтовому ящику (`imap.mail.ru`) и читает входящие письма.
- Находит письма по заданным триггерам (тема/адрес/текст, формат даты, период).
- Нормализует название контрагента и определяет нужный столбец периода.
- Вносит даты в Google-таблицу, обновляет комментарии и цветовые пометки повторов.
- Поддерживает ручную синхронизацию и синхронизацию по расписанию.

## Основные сценарии

1. Пользователь настраивает почту, таблицу и триггеры в интерфейсе.
2. При запуске синхронизации приложение получает последние `N` писем после выбранной даты.
3. Для каждого письма вычисляется запись вида: *контрагент + дата/период + лист/колонка*.
4. Изменения отправляются в Google Sheets пакетно.
5. Ошибки и служебные сообщения показываются в интерфейсе.

## Конфигурация

Локальные данные хранятся вне репозитория (директория `DATA/`, локальные `.env`/credentials).

Пример переменных окружения:

- `MAIL_SERVER`
- `MAIL_ADDRESS`
- `MAIL_PASSWORD`
- `SPREADSHEET_ID`
- `SPREADSHEET_CREDENTIALS`

## Структура проекта

- `main.py` — запуск приложения, orchestration синхронизации, интеграция UI.
- `mail.py` — работа с IMAP и представление письма.
- `spreadsheets.py` — операции с Google Sheets API.
- `modules/__init__.py` — общие импорты и утилиты.
- `tests/test_unit.py` — unit-тесты ключевых функций.

## Ограничения

- Основной сценарий рассчитан на Mail.ru IMAP.
- Для Google Sheets требуется корректный сервисный аккаунт и выданный доступ к таблице.
- Desktop-режим и сборка ориентированы на Windows.
